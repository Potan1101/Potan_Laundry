Step 1: Deskripsi Sistem

Sistem Informasi Laundry adalah platform digital untuk membantu admin dalam mengelola data laundry pelanggan. Sistem ini mencakup input data, perhitungan otomatis, serta status laundry seperti "Proses".

Step 2: Identifikasi Aktor

User

  • Register akun
  • Login
  • Melihat status laundry

Admin

  • Login
  • Input data laundry
  • Mengelola data

Step 3: Use Case

  • Melihat Data

Step 4: Hubungan

Step 9: Refleksi

- Saya memilih potensi lokal laundry untuk dikembangkan secara digital karena usaha laundry sangat dibutuhkan masyarakat sehari-hari. Sistem digital membantu pengelolaan menjadi lebih cepat dan rapi.

- Tantangan yang dihadapi adalah keterbatasan pemahaman teknologi serta keamanan data.

- Sistem sebaiknya dikelola oleh admin yang memahami komputer agar berjalan dengan baik.

- Pengembangan selanjutnya dapat berupa notifikasi, laporan keuangan, dan sistem online.
